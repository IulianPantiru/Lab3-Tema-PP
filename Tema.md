Problema 1.
Programul contine 2 clase: Item, care stocheaza informatiile unui articol(titlu, link, descriere si data publicarii), si Feed care reprezinta intregul feed RSS, care contine informatiile generale ale feed-ului(titlu, link, descriere si o lista de articole/Iteme). Programul se foloseste de biblioteca JSoup pentru a descarca si parsa feed-ul RSS. Se selecteaza primul element "channel" si se extrag informatiile generale ale feed-ului(titlu, link si descriere). Se selecteaza si se parcurg toate elementele de tip "item" in "channel", se extrag informatiile pentru Item(titlu, link, descriere si data publicarii) si se creaza obiecte de tip "Item". Aceste obiecte sunt apoi adaugate in Feed, care stocheaza intregul continut al Feed-ului. La final se afiseaza titlul si link-ul fiecarui Item din Feed-ul RSS.

Problema 2.
Programul defineste o functie Python ca sir de caractere care prelucreaza textul. Programul deschide fisierul in.txt (ebook-ul) si citeste tot continutul lui, inlocuieste spatiile multiple cu un singur spatiu, newline-urile le inlocuieste cu un singur newline si elimina liniile care contin doar cifre (numarul de pagina), cu ajutorul regex-urilor. Se creeaza un builder pentru contextul GraalVM, se evalueaza si se executa codul Python din Stringul Java.
